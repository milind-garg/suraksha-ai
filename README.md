# 🛡️ Suraksha AI

> AI-powered insurance intelligence platform for Indian families

Built for **AWS AI for Bharat Hackathon**

## Tech Stack
- **Frontend:** Next.js 14, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** AWS Lambda, Node.js 20.x, TypeScript
- **AI/ML:** AWS Bedrock (Claude 3.5 Sonnet), AWS Textract
- **Database:** AWS DynamoDB
- **Storage:** AWS S3
- **Auth:** AWS Cognito

## Features
- 📄 Upload insurance policy documents
- 🤖 AI-powered policy analysis in Hindi & English
- 🔍 Coverage gap identification
- 📊 Claim success probability prediction
- 💡 Personalized recommendations
